# 手续费

### 交易费用说明

在交易发生时，订单金额会扣除一定的手续费，剩余部分将添加到您的余额中。

### 平台费

使用 Stallee 进行销售时，我们将收取一小部分费用，称为平台费：

* 每笔交易我们收取 **6% + $0.50**

### 支付处理费

Stallee 的支付由 Stripe 和 PayPal 提供，各自会收取一定费用：

* Stripe **3.9% + $0.30**
* PayPal **4.4% + $0.30**

### 结算费

我们通过 PayPal 进行结算，每笔交易将收取 **1%** 的费用。
